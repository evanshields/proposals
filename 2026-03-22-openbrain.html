OpenBrain — Shieldstone's Persistent AI Memory System

What Is OpenBrain?

OpenBrain is Shieldstone's self-hosted, agent-readable memory system. It gives every AI tool — primarily Claude Code — persistent memory across all 8 business units, all email accounts, Google Drive, call recordings, and Evan's professional network.

The core idea: Every thought captured, every decision logged, every meeting noted, every email synced becomes permanently searchable by meaning — not just keywords. When you search "What do we know about Gateway?", OpenBrain finds relevant thoughts, emails, deals, contacts, and meeting notes across every table, ranked by semantic similarity.

The result: AI sessions never start from zero. Claude Code has full context of Shieldstone's operations, relationships, decisions, and communications — compounding over time as more data flows in.

Architecture

            + pgvector  API     API
            (VPS)       (3 accts) (3 accts)

Infrastructure:

  • Database: PostgreSQL 17 + pgvector 0.8.0 on Hostinger VPS
  • Embeddings: Ollama + nomic-embed-text (768 dimensions) — runs locally on VPS, zero API cost
  • Connection: SSH tunnel from your machine to VPS (port 443)
  • Google: OAuth2 tokens for all 3 accounts (Gmail, Drive, Calendar, Contacts — read-only)
  • Cost: ~$0.60/month (GPT-4o-mini for metadata extraction only)

How It Works Day-to-Day

Capturing Information

During any Claude Code session, information flows into OpenBrain naturally:

You: "I just got off a call with Charles Frazier about the GS Residential
      Fund I structure. He wants to target $50M first close by Q3."

Claude: [calls brain_capture with this thought]
        [calls brain_add_relationship to update Charles's last_contact]
        [calls brain_track_deal to update GS Residential Fund I]

Three tables updated from one conversation. All searchable forever.

Retrieving Information

You: "What do we know about Royal American?"

Claude: [calls brain_cross_reference "Royal American"]
        → Finds: relationship record, 3 related thoughts, 2 deal references,
          5 emails mentioning Royal American, 1 meeting note
        → Returns everything ranked by relevance

Proactive Insights

You: "Anything I'm forgetting about this week?"

Claude: [calls brain_upcoming_deadlines]
        → 2 overdue filings, 1 insurance renewal in 5 days
        [calls brain_stale_relationships]
        → James Chen (Royal American) — 45 days, needs outreach
        [calls brain_action_items]
        → 7 open items from this week's meetings

How to Use OpenBrain

Prerequisites

  1. SSH tunnel must be running (connects your machine to the VPS database + Ollama): 
    ssh -N -f -p 443 -i ~/.ssh/id_ed25519 -L 5433:localhost:5432 -L 11435:localhost:11434 -o ServerAliveInterval=60 root@187.124.113.118
  2. OpenBrain MCP is registered in Claude Code settings.json — it loads automatically at session start.

For Evan (Claude Code)

OpenBrain tools are available in every Claude Code session. Just talk naturally:

  • "Save this thought: [anything]" → brain_capture
  • "Search my brain for [topic]" → brain_search
  • "What's overdue?" → brain_upcoming_deadlines
  • "Search my email for [topic]" → gmail_search
  • "Find the file about [topic]" → gdrive_search

For Team Members (Chuck, Fahd, Alton)

Any team member with Claude Code (or any MCP-compatible AI tool) can connect to OpenBrain:

  1. Get SSH access to the VPS (ask Evan to add their SSH key)
  2. Run the SSH tunnel from their machine
  3. Add the OpenBrain MCP server to their Claude Code settings: 
    "mcpServers": {
  "openbrain": {
    "type": "stdio",
    "command": "node",
    "args": ["PATH_TO/openbrain/mcp-server/dist/index.js"],
    "env": {
      "DATABASE_URL": "postgresql://openbrain:ob-sh13ldst0ne-2026@localhost:5433/openbrain",
      "OLLAMA_URL": "http://localhost:11435"
    }
  }
}
  4. For Google Workspace access, they'll need their own OAuth tokens (run the authorize script)

The database is shared — everyone reads and writes to the same brain. Chuck logs a meeting note, Evan can search it instantly.
